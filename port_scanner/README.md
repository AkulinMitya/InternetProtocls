# UDP & TCP Port Scanner
**Ссылка на YouTube**  
  https://www.youtube.com/watch?v=2p1wPr33AW4
  
**Сканер TCP и UDP портов.**  

**Запуск**:  
py scanner.py hostname port_from port_to  

**Примечание**:  
hostname - имя хоста  
port_from и port_to - натуральные числа задающие диапазон сканирования
